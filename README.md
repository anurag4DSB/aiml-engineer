# aiml-engineer

Personal working repo for AI/ML learning: course notes and hands-on code, organized by course.

## Structure

```
courses/
  <specialization-or-course-name>/
    README.md          # source, syllabus links, progress tracker
    01-<course-name>/
      README.md
      notes/                        # concept notes, one file per week, in my own words
        week-01-<topic>.md
        week-01-<topic>.pdf         # optional: handwritten notes, same basename as the .md
      code/                         # my own notebooks, one subfolder per week
        week-01/
          <topic>-practice.ipynb    # built from scratch, not a copy of any course file
    02-<course-name>/
      ...
```

Each course gets its own folder. Multi-course specializations get a parent folder with one subfolder per course inside.

## This repo is public

This repo lives on GitHub as public. Course providers (DeepLearning.AI included) generally don't allow redistributing their lab/assignment code, even privately-then-later-public, so **nothing downloaded from a course goes in this repo** — only notes and practice code written from scratch.
